# ipl-data-analysis-python
IPL Data Analysis using Python
Project Overview

This project analyzes Indian Premier League (IPL) cricket data to extract useful insights about team performance, toss decisions, venues, and player achievements. The analysis is performed using Python data science libraries such as Pandas and Matplotlib.

The objective of this project is to demonstrate how data analysis techniques can be used to understand patterns and trends in sports data.

Dataset

The project uses publicly available IPL datasets containing match information.

Files used

matches.csv – Contains match level information such as teams, venue, winner, toss result, and player of the match.

deliveries.csv – Contains ball-by-ball information for each match.

Important columns in matches dataset

season

team1

team2

toss_winner

toss_decision

winner

venue

player_of_match

Key Insights

Some important insights obtained from the analysis:

Certain teams consistently dominate IPL seasons with higher win counts.

Winning the toss can have a measurable impact on the match outcome.

Some venues host significantly more matches compared to others.

Player of the match awards highlight consistently high-performing players.
